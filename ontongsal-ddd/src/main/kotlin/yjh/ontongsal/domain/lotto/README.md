
### 도메인 설계
- 책임 & 일관성 집합 기준으로 애그리거트 나누기.
- 애그리거트를 나누기 어렵다면 경험상 "이벤트 스토밍"을 해보면 도출하기 쉬워진다.

### 요구사항
- 로또 응모권
  - [ ] 응모권을 받을 수 있다.
    - 다양한 경로를 통해 응모권을 얻을 수 있다.(ex 결제, 이벤트 쿠폰, 게임 아이템 사용 등...)
- 로또(회차)
  - [ ] 로또 회차를 등록할 수 있다
  - [ ] 현재 진행중인 회차가 존재하면 등록할 수 없다.
  - [ ] 진행 회차 진행 기간은 종료일이 시작일보다 뒤여야 한다.
- 로또 응모
  - [x] 로또 번호의 숫자 범위는 1 ~ 45 이다.
  - [x] 로또 응모권은 6개의 중복 되지 않는 숫자이다.
  - [ ] 로또에 응모할 수 있다.
    - [ ] 로또 회차 진행중일때에만 응모할 수 있다.
    - [ ] 로또 회차 활성화일때에만 응모할 수 있다.
    - [ ] 응모권은 1장을 사용하여 로또에 응모할 수 있다.

- 로또 추첨
  - [ ] 당첨 번호를 추첨할 수 있다.
    - [ ] 당첨 번호는 로또 회차 진행이 끝나야만 추첨할 수 있다.
    - [ ] 당첨 번호 추첨 시 중복 되지 않는 숫자 6개와 보너스 번호 1개를 뽑는다.
  - [ ] 당첨은 1등부터 5등까지 있다.
    - [ ] 1 ~ 5등에 대한 보상 비율은 관리자가 설정할 수 있다.
    - [ ] 1등: 6개 번호 일치 / 총 상금 70%
    - [ ] 2등: 5개 번호 + 보너스 번호 일치 / 총 상금 20%
    - [ ]3등: 5개 번호 일치 / 총 상금 5%
    - [ ] 4등: 4개 번호 일치 / 총 상금 3%
    - [ ] 5등: 3개 번호 일치 / 총 상금 2%
